Exponential Metrics Modeling

Modeled metrics solver generates metrics through an exponential model according to the range projection. Model determines whether metrics is supposed to be displayed in term of ten exponent. Are we building large range that can be defined as positive exponent or small number that can be defined as negative exponent ? Exponent model generates metrics coming from this order-of-magnitude.
