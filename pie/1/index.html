Let's Make a Pie Chart

Sébastien Janaud - May 17, 2015 / JenScript

Pie Feature

A Pie chart displays value data as percentages of the whole. Categories are represented by individual slices. Pie charts are functionally identical to doughnut charts. Pie displays category groups, series groups, and values series as slices. JenScript does support Pie and common properties such as effect, label, listener. Register plugin PiePlugin in view projection. Add Pie in plugin then create PieSlice as parts of pie.

Learn :javascript pie plugin source

First step is to create the view which is the end chart. name attribute refers the root element where view chart is attached. 


<div style="padding-top : 40 px;float:left;" id="pieview"></div>

var view = new JenScript.View({
		name : 'pieview',
		width : 600,
		height : 300,
});

Now, create simple projection. By default pie is projected in user coordinate system and its default center is (0,0). it is drawn in center. in different context, you should want to draw pie in another location or in pixel coordinate at fixed position. 


var proj = new JenScript.LinearProjection({
		name : "proj1",
		minX : -1,
		maxX : 1,
		minY : -1,
		maxY : 1
	});

//or use identity projection
//var proj = new JenScript.IdentityProjection();
view.registerProjection(proj);

Then register pie chart plugin, and a pie with properties like radius, start angle, fill and effects.


var piePlugin = new JenScript.PiePlugin();
proj.registerPlugin(piePlugin);

var pie = new JenScript.Pie();
piePlugin.addPie(pie);

var fill = new JenScript.PieDefaultFill();
pie.setFill(fill);

var fx0 = new JenScript.PieLinearEffect();
pie.addEffect(fx0);
var fxl = new JenScript.PieReflectionEffect();
pie.addEffect(fxl);

Finally slice parts of pie.


var s1 = new JenScript.PieSlice({
	name : "s1",
	value : 45,
	themeColor : 'rgb(240,240,240)'
});
var s2 = new JenScript.PieSlice({

var s3 = new JenScript.PieSlice({

var s4 = new JenScript.PieSlice({
	name : "s4",

var s5 = new JenScript.PieSlice({
