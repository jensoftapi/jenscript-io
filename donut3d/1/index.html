Let's Make a Donut 3D Chart

Sébastien Janaud - May 8, 2015 / JenScript

Donut D3 Feature

A doughnut chart displays value data as percentages of the whole. Categories are represented by individual slices. A little more bit rock'n roll, Donut3D charts are functionally identical to pie charts. Donut3D displays category groups, series groups, and values series as slices.

JenScript does support donut/doughnut 3D and common properties such as effect, label, listener. Register plugin Donut3DPlugin in view projection. Add Donut3D in plugin then create Donut3DSlice as parts of donut.

GitHub donut3D plugin source

First step is to create the view which is the end chart. name attribute refers the root element where view chart is attached. 


<div style="padding-top : 40 px;float:left;" id="donut3dview"></div>

var view = new JenScript.View({
		name : 'donut3dview',
		width : 600,
		height : 300,
});

Now, create simple projection. By default donut is projected in user coordinate system and its default center is (0,0). it is drawn in center. in different context, you should want to draw donut in another location or in pixel coordinate at fixed position. 


var proj = new JenScript.LinearProjection({
		name : "proj1",
		minX : -1,
		maxX : 1,
		minY : -1,
		maxY : 1
	});

//or use identity projection
//var proj = new JenScript.IdentityProjection();
view.registerProjection(proj);

Then register donut3d plugin, and a Donut with properties like inner and outer radius, thickness, start angle and tilt.


var donut3DPlugin = new JenScript.Donut3DPlugin();
proj.registerPlugin(donut3DPlugin);

var donut = new JenScript.Donut3D({innerRadius:100,outerRadius:150,thickness : 60, startAngle : 300, tilt:40});
donut3DPlugin.addDonut(donut);

Finally slice parts of donut.


var s1 = new JenScript.Donut3DSlice({
	name : "s1",
	value : 45,
	themeColor : 'rgb(250, 250, 250)'
});
var s2 = new JenScript.Donut3DSlice({
	name : "s2",
	value : 5,
	themeColor : 'rgb(244, 145, 26)'
});
var s3 = new JenScript.Donut3DSlice({
	name : "s3",
	value : 30,
	themeColor : 'rgb(78, 148, 44)'
});
var s4 = new JenScript.Donut3DSlice({
	name : "s4",
	value : 5,
	themeColor : 'rgb(208, 58, 47)'
});

donut.addSlices([s1,s2,s3,s4]);
